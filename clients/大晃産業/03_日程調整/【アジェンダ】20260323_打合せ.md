# 打合せアジェンダ

**日時:** 2026年3月23日（月）10:00〜
**場所:** 大晃産業様
**出席者:** 大晃産業（桑田様、萩原様ほか） / えんのした（川路）

---

## 1. 取り組みの温度感確認（15分）

- 参加する方々（塾生・コーディネーター）の反応・期待感はどうか
- 社長をはじめとした幹部の皆様方の、本取り組みに対する熱量
- 1年目の方針確認：がっつり深くやるのか、まずは薄く広くやるのか
- 上記を踏まえた研修全体のトーン設定

## 2. コンテンツの方向性確定（20分）

- 全12回カリキュラムの概要説明・認識合わせ
- 現状の流れに沿うか、より経営寄りの内容にシフトするか
  → 経営寄りにする場合：会社の財務（BS/PL等）を研修コンテンツに組み込む
- 先方が特に重視したいテーマ・期待する成果
- 各回の内容に対する要望・カスタマイズの有無
- 研修のゴールイメージの共有
- **上記の確認をもって、コンテンツを確定させる**

## 3. 経営理念・役割行動・360度評価（15分）

- 経営理念の確認・共有
- 管理者に求められる役割行動の定義
  → 経営理念から落とし込んだ具体的な行動指標を設定
- 役割行動を基にした360度評価への接続
  → 評価項目の設計方針・実施時期の確認
- 研修プログラム全体を通じた評価の位置づけ

## 4. 最終報告の基準・目標共有（10分）

- 最終報告（第12回）の評価基準を提示
- 他社の最終報告イメージ（サンプル）を共有
- 自社でこの内容・レベルを目指してよいか確認
- 確認の上で、基準とサンプルを塾生に展開する流れの合意

## 5. 日程の最終確定（10分）

- 先方から回答いただいた確定日程の最終確認
- 各回の曜日・間隔に無理がないか確認
- 会場（社内会議室 or 外部）の確保状況

## 6. 参加メンバー・事務局の確定（15分）

- **事務局の選定**: 社内で事務局担当者を決めていただきたい
  → 連絡窓口・日程調整・資料配布等の実務を担当
  → メンバー構成表に事務局を追加
- **塾長（1名）**: 誰が就任するか（経営層・役員クラス）
- **コーディネーター（4名）**: 部長クラスの人選確認
- **塾生（12名）**: 課長・次長クラスの人選確認
- メンバー構成表の記入・返送の期日確認（未返送の場合）
- 各塾生の特徴・スキル・育成課題のヒアリング

## 7. チーム編成の方針（10分）

- 4チーム×3名の編成方針（部署・年齢・スキルのバランス）
- コーディネーターと塾生の組み合わせの考え方
- 編成案はこちらで作成し、後日提示でよいか

## 8. 参加者のスキル確認（10分）

- 塾生のPCスキル（特にPowerPoint）のレベル感
  → 第6回「プレゼンテーション技術」の内容レベル調整に反映
- Excel・Word等の基本スキルの把握
- プレゼン経験の有無

## 9. 課題図書について（10分）

- 課題図書の提示可否（費用負担含む）
- 4回分程度を想定：どのタイミングで指定するか
- 選書はえんのした側で行い、先方承認の流れでよいか
- 購入・配布の方法（会社負担 or 個人負担）

## 10. 振り返りシートの運用方法（10分）

- 振り返りシート（A3 Excel）の内容確認
- 運用フロー：研修当日記入 → 実践 → 次回までに実践振り返り記入
- **上司確認の仕組み**: 誰が確認するか（直属上司 or コーディネーター）
- 上司フィードバックの運用ルール
- 提出・回収の方法（紙 or データ）

## 11. 案内書の確認（5分）

- 塾生向け案内書の内容確認・修正点
- コーディネーター向け案内書の内容確認・修正点
- 配布時期・配布方法の決定

## 12. 合宿・講師移動について（15分）

- **合宿（第11回）の実施可否の最終判断**
- 候補地の検討（先方からの提案 or こちらから候補提示）
- 予算感の確認（宿泊費・会場費・食事代）
- 1泊2日のスケジュール（DAY1: 9:00〜17:00 / DAY2: 8:30〜15:00）
- 移動手段（バス手配 or 各自 or 社用車）
- **講師の前泊について**
  → 尾道で朝9時開始の場合、移動トラブルを考慮し前泊させていただきたい
  → 新幹線での移動を想定
  → 前泊の可否・費用負担の確認

## 13. 上司向けオリエンテーション（5分）

- 4月中旬に30分・無料で実施予定
- 対象者：塾生の直属上司
- 日程候補の調整方法
- オンライン or 対面の確認

## 14. 今後のスケジュール確認（5分）

| 時期 | タスク |
|------|--------|
| 3月末 | メンバー構成表の最終確定（事務局含む） |
| 3月末 | 経営理念・役割行動の素案作成 |
| 4月上旬 | 振り返りシート最終版作成 |
| 4月上旬 | 課題図書の選定・発注 |
| 4月上旬 | 最終報告の基準・サンプル資料の送付 |
| 4月中旬 | 上司向けオリエンテーション実施 |
| 4月中旬 | 360度評価の設計・項目確定 |
| 5月上旬 | 第1回研修資料の完成 |
| 5月中旬 | 参加者への最終案内・事前課題送付 |
| 6月 | 第1回研修「オリエンテーション＆管理職の役割認識」 |

## 15. その他・質疑応答（5分）

- 研修当日の受付・準備（プロジェクター、ホワイトボード等）
- 写真撮影・記録の可否
- 緊急連絡先・窓口の確認
- 次回の連絡タイミング
