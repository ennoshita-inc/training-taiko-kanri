# 打合せアジェンダ

**日時:** 2026年3月23日（月）10:00〜
**場所:** 大晃産業様
**出席者:** 大晃産業（桑田様、萩原様ほか） / えんのした（川路）

## 1. 取り組みの温度感確認（15分）

- 参加する方々（塾生・コーディネーター）の反応・期待感はどうか
- 社長をはじめとした幹部の皆様方の、本取り組みに対する熱量
- 1年目の方針確認：がっつり深くやるのか、まずは薄く広くやるのか
- 上記を踏まえた研修全体のトーン設定

## 2. コンテンツの方向性確定（20分）

- 全12回カリキュラムの概要説明・認識合わせ
- 現状の流れに沿うか、より経営寄りの内容にシフトするか
  → 経営寄りにする場合：会社の財務（BS/PL等）を研修コンテンツに組み込む
- 先方が特に重視したいテーマ・期待する成果
- 各回の内容に対する要望・カスタマイズの有無
- 研修のゴールイメージの共有
- **上記の確認をもって、コンテンツを確定させる**

## 3. 360度評価の進め方（15分）

- 現状、経営理念・役割行動・人事評価制度が未整備であることを前提に
- **提案：研修で使用するコンピテンシー項目を、そのまま360度評価の指標として活用**
  → 研修で学ぶ内容と評価が直結し、塾生の成長を可視化できる
- コンピテンシー評価項目（6カテゴリ・19項目）の確認
  - **A. 分析・構想力**: 情報把握力 / 問題分析力 / 課題設定力
  - **B. 実行・推進力**: 計画策定力 / プロセス管理力 / 決断力
  - **C. 対話・共感力**: ヒアリングスキル / プレゼンテーションスキル / 対人感受性 / 自己開示能力
  - **D. 組織牽引力**: 指導育成力 / 組織活性化力 / 方向づけする力
  - **E. 自己基盤力**: 自律一貫性 / ストレス耐性 / 倫理観
  - **F. 変革・挑戦力**: 柔軟性 / バイタリティ / 革新性
- 評価の実施時期：研修開始前（6月）と研修終了後（翌年6月）の2回を想定
- 評価者の範囲：上司・同僚・部下（＋自己評価）
- 将来的に経営理念・役割行動が整備された際に、評価項目を再設計する流れ

## 4. 最終報告の基準・目標共有（10分）

- 最終報告（第12回）の評価基準を提示（添付資料あり）
- **他社事例の評価基準（5カテゴリ・24項目）を参考として共有**
  - **I 目指す姿の設定**: 期待に対する関心度 / 想いの強さ / 目標の明確さ / 目標設定の速さ
  - **II 現状の把握**: 論点の適切さ / 問題点の妥当性 / 原因追及の妥当性 / データの収集や活用の程度
  - **III 打ち手の設定**: 打ち手の数の多さ / 優先順位の意識の程度 / 計画性の程度
  - **IV コンピテンシーの発揮**: 行動の種類の多さ / 省察の程度 / 成果
  - **V 成果の発表**: 準備の程度 / スライドの出来栄え / 実演時の出来栄え / フィードバックの受容度
- 自社でこの内容・レベルを目指してよいか確認
- 確認の上で、基準とサンプルを塾生に展開する流れの合意

## 5. 日程の最終確定（10分）

- 先方から回答いただいた確定日程の最終確認
- 各回の曜日・間隔に無理がないか確認
- 会場（社内会議室 or 外部）の確保状況

## 6. 参加メンバー・事務局の確定（15分）

- **事務局の選定**: 社内で事務局担当者を決めていただきたい
  → 連絡窓口・日程調整・資料配布等の実務を担当
  → メンバー構成表に事務局を追加
- **塾長（1名）**: 誰が就任するか（経営層・役員クラス）
- **コーディネーター（4名）**: 部長クラスの人選確認
- **塾生（12名）**: 課長・次長クラスの人選確認
- メンバー構成表の記入・返送の期日確認（未返送の場合）
- 各塾生の特徴・スキル・育成課題のヒアリング

## 7. チーム編成の方針（10分）

- 4チーム×3名の編成方針（部署・年齢・スキルのバランス）
- コーディネーターと塾生の組み合わせの考え方
- 編成案はこちらで作成し、後日提示でよいか

## 8. 参加者のスキル確認（10分）

- 塾生のPCスキル（特にPowerPoint）のレベル感
  → 第6回「プレゼンテーション技術」の内容レベル調整に反映
- Excel・Word等の基本スキルの把握
- プレゼン経験の有無

- 課題図書の提示可否（費用負担含む）
- 4回分程度を想定：どのタイミングで指定するか
- 選書はえんのした側で行い、先方承認の流れでよいか
- 購入・配布の方法（会社負担 or 個人負担）
- **現時点での課題図書案（4冊）**
  1. 川上真史『会社を変える社員はどこにいるか』2003年、ダイヤモンド社
     → 研修で伝えたいコンピテンシーについて優しく書かれています
     （期限：7/末まで）RTシェア予定：8/7
  2. 嶋田毅『実況ロジカルシンキング教室』2011年、PHP研究所
     → 研修で伝えたい問題解決のステップや、ロジカルシンキングが優しく書かれています
     （期限：9/末まで）RTシェア予定：10/9
  3. 細谷功『「具体⇄抽象」トレーニング 思考力が飛躍的にアップする29問』2020年、PHPビジネス新書
     → 研修の中で度々口にする論点です。選抜教育Ⅱの参加者には絶対に押さえて頂きたい
     （期限：年始まで）RTシェア予定：1/22
  4. 入山章栄『世界標準の経営理論』2019年、ダイヤモンド社

     （期限：3/末まで）RTシェア予定：4/13

## 12. 合宿・講師移動について（15分）

- **合宿（第11回）の実施可否の最終判断**
- 候補地の検討（先方からの提案 or こちらから候補提示）
- 予算感の確認（宿泊費・会場費・食事代）
- 1泊2日のスケジュール（DAY1: 9:00〜17:00 / DAY2: 8:30〜15:00）
- 移動手段（バス手配 or 各自 or 社用車）
- **講師の前泊について**

  → 新幹線での移動を想定
  → 前泊の可否・費用負担の確認
