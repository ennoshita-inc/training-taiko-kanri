# 04_参加者管理

## メンバー構成

| 役割 | 人数 | 対象クラス | 備考 |
|------|------|-----------|------|
| 塾長 | 1名 | 経営層・役員 | 参加可能な回にオブザーブ出席 |
| コーディネーター | 4名 | 部長クラス | 各チームのリーダー役（塾生3名を担当） |
| 塾生 | 12名 | 課長・次長クラス | 4チーム×3名 |
| **事務局** | **要選定** | **－** | **連絡窓口・日程調整・資料配布等の実務担当** |

## 事務局について（3/23打合せで確認）

- 社内で事務局担当者を選定いただく
- 役割：連絡窓口、日程調整、資料配布、振り返りシートの回収等
- メンバー構成表に事務局を追加する
